# Лабораторная работа 6, вариант 22

Задание: заменить расширение файлов, размер которых меньше заданного, на первые буквы имени файла

## Как работает

Скрипт идёт по папке через `find`, для каждого файла смотрит размер через `stat`.
Если меньше порога, берёт имя без расширения, режет его по `_` и `-`, берёт первую букву каждой части и склеивает в новое расширение. Потом переименовывает через `mv`.

`my_report.docx` -> `my_report.mr`, `hello.txt` -> `hello.h` и т.д.

Скрытые файлы, которые начинаются с `.`, не трогает.

## Запуск

```bash
./lab6.sh [-r] [-n] [-v] <dir> <size>
```

- `-r` - рекурсивно
- `-n` - dry run, ничего не переименовывает, просто показывает
- `-v` - выводить каждый rename
